# Platform Agent UI

A modern chat UI for Platform Agent powered by SurrealDB.

## Features

- 💬 Chat interface with streaming responses
- 🧠 Memory & conversation history
- 🔍 Vector search
- 🗃️ Knowledge graph visualization
- 📊 Real-time metrics

## Tech Stack

- Astro + React
- Tailwind CSS
- Vercel AI SDK

## Getting Started

```bash
cd ui
npm install
npm run dev
```

## Environment

```env
PUBLIC_API_URL=http://localhost:8000
PUBLIC_SURREALDB_WS=ws://localhost:8000/rpc
```
